#  Canteen Management System

A console-based **C++** application developed using **Object-Oriented Programming (OOP)** concepts. The system allows users to browse the menu, place food orders, calculate bills, and generate the final receipt.

##  Features

- Display food menu
- Order multiple items
- Calculate total bill
- Family Deal option
- Generate final bill

# 🛠 Technologies

- C++
- Object-Oriented Programming (OOP)
- Visual Studio

##  How to Run

1. Open the project in Visual Studio.
2. Build and run the project.
3. Follow the on-screen menu to place an order.
